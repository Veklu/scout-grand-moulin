# Scout Grand-Moulin — Theme WordPress

Un theme WordPress personnalise pour le 5e Groupe scout Grand-Moulin (Deux-Montagnes, Quebec). Concu pour les groupes Scouts du Canada avec accessibilite (WCAG 2.1 AA) et conformite a la Loi 25 du Quebec.

## Fonctionnalites

- **Design adaptatif** — Mise en page mobile-first optimisee pour les familles
- **Modeles de pages personnalises** — Unites, Inscription, Galerie, Agenda, Benevoles, Confidentialite, Conditions
- **Galerie photo** — Galerie par albums avec lightbox et categorisation
- **Consentement aux temoins** — Banniere de temoins conforme a la Loi 25
- **Accessibilite** — Conforme WCAG 2.1 AA avec liens d'evitement, attributs ARIA et navigation au clavier
- **Bloc personnalise** — Bloc bulle scout pour l'editeur Gutenberg

## Installation

1. Compresser le dossier `scout-grand-moulin` en `.zip`
2. WordPress : Apparence > Themes > Ajouter > Televerser
3. Activer le theme

## Configuration

1. **Logo** : Apparence > Personnaliser > Identite du site
2. **Menus** : Apparence > Menus > creer « Navigation principale »
   - Pour les boutons CTA : ajouter la classe CSS `menu-item-cta`
3. **Pages** : Creer chaque page et assigner le modele :
   - Accueil > page d'accueil statique (Reglages > Lecture)
   - Nos unites, Inscription, Galerie, Agenda, Notre equipe, Confidentialite, Conditions
4. **Coordonnees** : Apparence > Personnaliser > Coordonnees du groupe
5. **Confidentialite** : Reglages > Confidentialite > selectionner la page

## Internationalisation

Le theme supporte le francais (par defaut) et l'anglais. Pour changer de langue, modifier la locale WordPress via Reglages > General > Langue du site.
